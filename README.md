A quantum random-walk framework that reproduces Black–Scholes option prices with high accuracy. The project includes a production-ready pricer, analytical and Monte Carlo baselines, and full notebooks showing the derivation, implementation, and validation.

---

## Features

- Quantum random-walk discretization of the Black–Scholes PDE  
- Production-ready Python implementation (src/)  
- Black–Scholes analytics and Monte Carlo baselines  
- Configurable experiments (config/) and datasets (datasets/)  
- Jupyter notebooks for theory, diagnostics, and experiments (notebooks/)  
- Documentation and examples for quick adoption (docs/, documents/)
